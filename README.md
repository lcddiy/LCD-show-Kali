# How to install the driver
-------for 3.5inch GPIO--------------\
sudo rm -rf LCD-show-Kali\
git clone github.com/lcddiy/LCD-show-Kali.git\
chmod -R 755 LCD-show-Kali\
cd LCD-show-Kali\
sudo ./LCD35-show
